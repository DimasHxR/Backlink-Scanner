Kegunaan Utama:

1. Analisis Backlink Otomatis
 - Memindai website untuk menemukan semua link keluar (outbound links)
 - Menganalisis kualitas setiap link berdasarkan domain dan kontennya

2. Deteksi Link Berbahaya
 - Mengidentifikasi link mencurigakan seperti:
  - Casino/gambling
  - Pharmacy ilegal
  - Konten dewasa
  - Website scam/penipuan
  - Situs hacking/cracking

3. Scoring Sistem
 - Memberikan skor risiko (0-100) berdasarkan:
  - Keyword berbahaya yang ditemukan
  - Link tersembunyi (hidden links)
  - Jumlah link keluar yang mencurigakan
  - Kualitas domain link

4. Fitur Lengkap:
 - Batch Processing: Bisa scan multiple URLs sekaligus
 - Rate Limiting: Pembatasan 10 detik antar scan
 - Multi-format Export: TXT, CSV, JSON
 - Detail Reporting: Menampilkan detail lengkap setiap link
 - Quality Assessment: Membedakan link berkualitas vs berbahaya

Cara Kerja:

Proses Scanning:
 - Input URLs → User memasukkan daftar URL
 - Fetch Content → Mengambil konten HTML website
 - Extract Links → Menemukan semua link dalam halaman
 - Analyze → Memeriksa pola berbahaya, hidden elements, suspicious keywords
 - Scoring → Memberikan skor risiko
 - Reporting → Menghasilkan laporan lengkap

Deteksi Spesifik:
 - Hidden Links: CSS tricks (display:none, opacity:0, dll)
 - Suspicious Patterns: Regex patterns untuk konten ilegal
 - Domain Quality: .gov/.edu mendapat poin lebih
 - Keyword Matching: Mencocokkan dengan daftar keywords berbahaya

Penggunaan Khas:
 - SEO Specialist: Memeriksa profil backlink website
 - Website Owner: Mendeteksi hack/malware pada website
 - Content Manager: Memastikan tidak ada link spam dalam konten
 - Security Auditor: Audit keamanan website

Kode ini sangat berguna untuk audit SEO dan keamanan website, terutama untuk mendeteksi link spam atau backlink beracun yang bisa merusak ranking SEO atau mengandung malware.
